## Top 5 Linux Commands I Would Use First

1. ls -l
   - Checks files, permissions, owner, and group.

2. ps aux
   - Shows running processes.

3. systemctl status nginx
   - Checks whether a service is running.

4. journalctl -u nginx -n 20
   - Displays recent logs for troubleshooting.

5. df -h
   - Checks disk usage
